# 🔧 Zoho Mail SMTP Troubleshooting Guide

## Current Status

Your SMTP configuration is set up, but Zoho Mail authentication is failing.

## Steps to Resolve

### 1. Verify Zoho Mail Settings
Go to **Zoho Mail Settings** → **Accounts** → **SMTP/POP3 Credentials**

- ✅ Is SMTP access **enabled** for this account?
- ✅ Is the account an organization account (not just personal)?
- ✅ Are there any IP restrictions blocking the connection?

### 2. Verify the Password
The password `eskhVhLErF8b` might be:
- A database password (not SMTP password)
- An app-specific password that needs to be generated in Zoho
- Subject to account restrictions

**To generate/reset SMTP password in Zoho:**
1. Go to **Zoho Mail** → **Settings** → **Mail accounts**
2. Click on your account
3. Find **SMTP Settings** or **SMTP Credentials**
4. Look for option to "Generate new password" or "Reset password"
5. Copy the newly generated SMTP password
6. Update `.env.local` with the new password

### 3. Check Account Type
**For Paid Organization Accounts:**
```env
SMTP_HOST=smtppro.zoho.com  (Use this, not smtp.zoho.com)
SMTP_PORT=465
SMTP_USER=Hospitality.is2@shivai.co.in
SMTP_PASSWORD=<your-zoho-smtp-password>
```

**For Free/Personal Accounts:**
```env

SMTP_PORT=465
SMTP_USER=Hospitality.is2@shivai.co.in
```

## Alternative: Use Zoho API Instead of SMTP

If SMTP continues to fail, consider using Zoho's Email API:

```javascript
// Example with Zoho API (requires API token)
const axios = require('axios');

const sendEmailViaZoho = async (to, subject, body) => {
  await axios.post('https://www.zohoapis.com/crm/v2/send_email', {
    data: [
      {
        email: to,
        subject: subject,
        content: body
      }
    ]
  }, {
    headers: {
      'Authorization': `Zoho-oauthtoken ${process.env.ZOHO_API_TOKEN}`
    }
  });
};
```

## Common Zoho Mail SMTP Issues

| Error | Cause | Solution |
|-------|-------|----------|
| 535 Authentication Failed | Wrong password | Verify/regenerate SMTP password in Zoho settings |
| Connection timeout | Wrong host/port | Use `smtppro.zoho.com` for paid, `smtp.zoho.com` for free |
| Account locked | Too many failed attempts | Wait 30 minutes or reset password in Zoho |
| TLS error | Security settings | Ensure port 465 (SSL) and not 587 (TLS) |

## Need Help?

1. **Check Zoho Documentation**: https://www.zoho.com/mail/help/configurations/smtp-configuration.html
2. **Contact Zoho Support**: Through your Zoho Mail dashboard
3. **Verify Credentials**: Ensure you have the correct email and password
4. **Check Account Status**: Ensure the account is active and SMTP is enabled
